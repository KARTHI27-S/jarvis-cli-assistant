JARVIS CLI Assistant is a modular Python-based command-line assistant inspired by JARVIS,
designed for system monitoring, notes, reminders, and basic SOC analyst utilities.

---

## Features (v1.0.0)

- System status monitoring (CPU, RAM, Disk, OS)
- Show current date & time
- Notes management (add & view)
- Reminder system (runtime-based)
- SOC Analyst Dashboard (basic)
- Clean menu-driven CLI interface
